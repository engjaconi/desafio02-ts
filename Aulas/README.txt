Criado o projeto com: 
>npx create-react-app my-app --template typescript // Iniciando com template typescript
>npm start // Para rodar a aplicação

Para estilizar os componentes dentro do mesmo arquivo temos que importar um pacote.
> npm i style-componentes

Rodar testes unitários com:
>npm run testes

TDD = criar o teste antes do desenvolvimento, ir fazendo aos poucos. Teste, falha, implementa...
